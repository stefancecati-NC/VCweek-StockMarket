# Daily Stock Market Summary Application

A modern web application that provides AI-powered daily stock market summaries, real-time market data, and upcoming earnings reports. Built with Node.js, Express, and vanilla JavaScript with a beautiful, responsive UI.

## Features

- 🤖 **AI-Generated Daily Summaries** - Uses ChatGPT to provide intelligent market analysis and insights
- 📊 **Real-time Market Data** - Displays major market indices and stock prices
- 📈 **Top Movers** - Shows biggest gainers and losers of the day
- 📅 **Earnings Calendar** - Upcoming earnings reports for major companies
- 🎯 **Major Companies Tracking** - Focus on the most valuable companies
- 📱 **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- ⏰ **Automated Updates** - Daily summary generation at 8 AM EST
- 🔄 **Manual Refresh** - Refresh data on demand

## Tech Stack

- **Backend**: Node.js, Express.js
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI Integration**: OpenAI GPT-3.5 Turbo
- **Scheduling**: Node-cron
- **Styling**: Modern CSS with gradients and glassmorphism effects

## Quick Start

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- OpenAI API key (optional, works with mock data without it)

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd stock-market-daily-summary
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   ```
   Edit `.env` and add your API keys:
   ```env
   OPENAI_API_KEY=your_openai_api_key_here
   PORT=3000
   ```

4. **Start the application**
   ```bash
   npm start
   ```
   
   For development with auto-restart:
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:3000`

## API Endpoints

### GET `/api/daily-summary`
Returns the AI-generated daily market summary including sentiment analysis and key points.

### GET `/api/market-data`
Fetches current market overview including indices, top movers, and major stocks data.

### GET `/api/earnings-calendar?days=7`
Returns upcoming earnings reports for the specified number of days (default: 7).

### GET `/api/company-data/:symbol`
Gets detailed information about a specific company by stock symbol.

### POST `/api/refresh-summary`
Manually triggers a new daily summary generation.

## Configuration

### Environment Variables

- `OPENAI_API_KEY` - Required for AI-generated summaries (falls back to mock data if not provided)
- `ALPHA_VANTAGE_API_KEY` - Optional, for real stock data (currently uses mock data)
- `FINNHUB_API_KEY` - Optional, for real stock data (currently uses mock data)
- `PORT` - Server port (default: 3000)

### Customization

#### Adding New Companies
Edit `services/stockService.js` and modify the `majorCompanies` array:
```javascript
this.majorCompanies = [
    'AAPL', 'MSFT', 'GOOGL', 'AMZN', 'TSLA', 'META', 'NVDA',
    // Add your preferred companies here
];
```

#### Changing Update Schedule
Modify the cron expression in `server.js`:
```javascript
// Currently runs at 8 AM EST daily
cron.schedule('0 8 * * *', () => {
    generateDailySummary();
}, {
    timezone: "America/New_York"
});
```

#### Integrating Real Stock Data
The application is designed to work with mock data by default. To integrate real stock market APIs:

1. Sign up for API keys from providers like Alpha Vantage or Finnhub
2. Update the methods in `services/stockService.js` to make real API calls
3. Add your API keys to the `.env` file

## Troubleshooting

### Common Issues

1. **Application won't start**
   - Check that Node.js is installed (`node --version`)
   - Ensure all dependencies are installed (`npm install`)
   - Verify port 3000 is available

2. **AI summaries not generating**
   - Check that `OPENAI_API_KEY` is set correctly in `.env`
   - Verify OpenAI API key has sufficient credits
   - Check console logs for API errors

3. **Data not loading**
   - Check browser console for JavaScript errors
   - Verify API endpoints are responding (`/api/market-data`)
   - Check network connectivity
